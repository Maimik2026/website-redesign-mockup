
# Website Redesign – SP-Elektroniikka

# Overview
This project is a usability-focused redesign of the SP-Elektroniikka website.

The goal was to identify key UX problems and improve navigation, product discovery, and overall user experience.

# Problems Identified
- Long and confusing category lists
- Poor product discoverability
- No filtering options
- Too much scrolling
- Weak visual feedback (e.g. add-to-cart)
- Poor mobile usability

# Solutions
- Grouped categories into clearer structure
- Added filters (resistance, power, etc.)
- Improved product card layout
- Better image preview (zoom functionality)
- Clearer add-to-cart feedback
- Improved mobile navigation and layout

# Tools
- PowerPoint
- Figma (concept)
- UX analysis
